PHP Network Weathermap v0.95

Interactive Map Editor

Welcome to the new PHP Weathermap Editor. Just like the old editor, "This editor is not finished! There are many features of Weathermap that you will be missing out on if you choose to use the editor only".

Obviously the whole point of rewriting the editor is that there are less of these features, but there will still be some. If you need fine-control over some aspects of the map then you should consider reading the config reference, and breaking out Notepad/vi. It's not so scary.

For the best experience of the editor, you might want to maximise your browser window. The editor is most-tested in FireFox 2.0, but should work in most modern browsers.

Click anywhere on the text to get started!
